How to test?

Try to create real articles, publish them online and let us know. Tell us what you like about the interface and what you think could be improved. We have an open ear to your feedback and it's very possible that good suggestions will make it into the final product. To get in touch, email us contribute@substance.io. Try to keep the mails short and informal. We'll try to respond as quickly as possible.

Read the user manual

Make sure you read the user manual. We'll keep that document updated to describe new features, known issues and point you to useful resources.

Be aware of known issues

Do not use the OSX Edit menu yet. It is not connected properly. Instead use CMD+C and CMD+V for copy and paste. Also if you try to quit the entire application from the menu, some things get broken (e.g. the menu will no longer respond). We are working on a fix but in the meanwhile please just restart the application.
