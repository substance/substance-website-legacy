#

Neues Projekt (Web-Seite)

Legen Sie ein neues Projekt an. Ein Projekt entspricht einer Server-Instanz mit eigener URL, die z.B. für eine Web-Seite zuständig ist.
Dialog zur Erstellung eines neuen Servers (Projekt)
Wählen Sie anschließend den Projekt-Typ PHP 5.5 & MySQL aus.
Auswahl des Projekt-Typs.
Geben Sie einen Namen ein (z.B. Publikationen).
Dialog zur Konfiguration des Projekt-Namens.
Anschließend wird der Server angelegt.
Ihr Server ist online.

Datentransfer

Für den Transfer von Daten stehen verschiedene Programme zur Verfügung (z.B. WinSCP oder CyberDuck). Die nötigen Informationen zur Herstellung einer Verbindung können Sie dem Files Dialog entnehmen.
Informationen für die Herstellung einer SFTP/SSH Verbindung.
Die markierten Daten müssen in ein Daten-Transfer Programm eingegeben werden.
Konfiguration einer Verbindung in CyberDuck
